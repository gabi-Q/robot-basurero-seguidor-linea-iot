Requisitos del Proyecto Robot Basurero Seguidor de Línea IoT

1. Firmware Arduino (de platformio.ini y main.ino):
- Plataforma: espressif32
- Placa: esp32dev
- Framework: arduino
- Dependencias:
  - mobizt/Firebase ESP32 Client @ ^4.4.17
  - pololu/VL53L0X @ ^1.3.1
  - madhephaestus/ESP32Servo @ ^3.0.8
  - martinsos/HCSR04 @ ^2.0.0
- Librerías estándar: Arduino.h, Wire.h, WiFi.h, WebServer.h, FirebaseESP32.h, VL53L0X.h, time.h

2. Aplicación Web React (de package.json, package-lock.json, tsconfig.json y App.tsx):
- Dependencias principales:
  - chart.js @ ^4.5.0
  - firebase @ ^11.9.1
  - react @ ^19.1.0
  - react-dom @ ^19.1.0
- Dependencias de desarrollo:
  - @types/node @ ^22.14.0
  - typescript @ ~5.7.2
  - vite @ ^6.2.0
- Configuración: TypeScript con target ES2020, module ESNext, jsx react-jsx
